API Documentation

Endpoints

1. User Registration

URL: /api/users/register

Method: POST

Request Body:

{
    "username": "your_username",
    "email": "your_email@example.com",
    "password": "your_password"
}

2. User Login

URL: /api/users/login

Method: POST

Request Body:

{
    "email": "your_email@example.com",
    "password": "your_password"
}

3. User Profile

URL: /api/users/profile

Method: GET

Authorization: Bearer token required

4. User Profile Delete

URL: /api/users/profile/delete

Method: POST

Authorization: Bearer token required

5. Edit User Profile

URL: /api/users/profile/edit

Method: POST

Authorization: Bearer token required

Request Body:

{
    "username":"your_username",
    "email": "your_email@example.com",
    "password": "your_password"
}
